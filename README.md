# Personal Dashboard

A clean, minimal personal dashboard built with Vanilla JavaScript.

## Features
- Add, edit, delete, and complete tasks
- Per-user dashboard using LocalStorage
- Light / Dark mode
- Accent themes (slate, red, purple, blue)
- Task filtering and search
- Progress tracking

## Tech Stack
- HTML
- CSS
- Vanilla JavaScript
